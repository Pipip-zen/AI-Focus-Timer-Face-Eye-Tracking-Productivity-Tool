# Changelog

All notable changes to this project will be documented in this file.

## [v1.1.0] - 2025-12-07

### Added

- Mobile/Web version using Streamlit and WebRTC
- Real-time camera streaming via browser
- Head pose detection on mobile
- Eye Aspect Ratio (EAR) based sleep detection
- On-screen focus status and timer overlay

### Improved

- Better modular structure for video processing
- More flexible sensitivity controls via sidebar

---

## [v1.0.0] - Initial Release

### Added

- Desktop version with OpenCV webcam input
- Real-time face tracking using MediaPipe
- Head pose detection (pitch & yaw)
- Sleep detection using eye aspect ratio (EAR)
- Smart focus-based Pomodoro timer
- Sound alerts using Windows winsound
- Manual calibration, pause and resume
